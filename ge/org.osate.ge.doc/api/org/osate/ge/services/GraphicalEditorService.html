      • operateOn

        java.lang.Object operateOn​(java.lang.Object operand,
                           java.lang.Object operation)
        Activates a specified operation and returns the result of that operation.
        Parameters:
        operand - is the object on which to activate the operation.
        operation - is the operation to be performed. The operation must have a method annotated with the Activate annotation.
        Returns:
        the result of the operation. Returns null if the object could not be operated on. The operation's activation method is a called with the standard set of objects available for injection. The following named values are supported: Names.DIAGRAM_BO - The business object of the diagram in which the operand is contained. May be null. Names.BUSINESS_OBJECT - The business object represented by the operand.

        canOperateOn(Object)
