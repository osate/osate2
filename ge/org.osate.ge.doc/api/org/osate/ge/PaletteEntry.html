Package org.osate.ge

Interface PaletteEntry

  • public interface PaletteEntry
    An entry in the graphical editor palette. Palette entries are provided to the graphical editor by business object handlers via a method with the GetPaletteEntries annotation. PaletteEntry objects must be created using the PaletteEntryBuilder class.
    See Also:
    GetPaletteEntries, PaletteEntryBuilder
