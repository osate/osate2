Package org.osate.ge.di

Annotation Type GetNameForUserInterface

  • @Documented
@Target(METHOD)
@Retention(RUNTIME)
public @interface GetNameForUserInterface

    This annotation must not be applied to more than one method per class.

    Usages

    UsageDescriptionReturn Value
    Business Object HandlerReturns the name that should be displayed in the user interface for an object. If a handler does not implement a method with this annotation, the value retrieved using GetName will be used.String

    Named Parameters

    ParameterUsageDescription
    Names.BUSINESS_OBJECTBusiness Object handlerThe business object for which to return the name.
