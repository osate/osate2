4.8 Synchronizing Eclipse/OSATE with the File System
This capability is part of the Eclipse functionality.  You may have edited a file that is in an Eclipse workspace from outside Eclipse—either with an editor or by (re-)generating the file with a tool outside Eclipse.  Eclipse will detect that changes have been made externally.  You will have to perform a “Refresh” (in the “File” menu or Eclipse Navigator context menu) in the Navigator for Eclipse to update its state information of the file system.
You can also set an option to automatically perform a refresh if changes happen to the file system. You can do so in the “General > Workspace” dialog pane of the “Window > Preferences…” dialog.
