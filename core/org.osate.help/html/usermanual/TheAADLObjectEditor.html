7. The AADL Object Editor
The AADL Object Editor can be invoked in one of two ways:
    • By opening an existing resource the “aaxl” extension. These resources are represented by the Aqua colored Aadl icon as shown below.
    • By creating a new object model resource using the Aadl wizards; see Section 4.4.
