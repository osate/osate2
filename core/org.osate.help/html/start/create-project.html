Getting started: create a project

To create an AADL project, go to the File menu, then New and Others as in the following picture.

A new dialog would appear to select your project type. Select Aadl Wizard and then Aadl project, as shown in the following picture.

A new dialog appears, enter your project name, as in the following picture.

Finally, the project is created. You can then write your models, the next section details how to proceed.
