Resources and Copyright

AADL on the WWW

For people that would like to learn more about AADL and contribute to the community, the following online resources are useful:

  • The official AADL Wiki for all documentation and scientific publication about AADL.
  • The SAE standard page
  • The Github development area

Mailing-list

The AADL community users has its own mailing-list (called sae-aadl-users). This is a good way to keep in touch with the community and asks potential questions.

Interested users can subscribe to the list on https://lists.sei.cmu.edu/mailman/listinfo/sae-aadl-users.
