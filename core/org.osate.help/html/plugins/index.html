OSATE plug-ins

OSATE also provide a set of plug-ins, either for validating your models or interface them with other tools. Users can develop their own plug-ins to connect AADL models with other tools. The following sub-sections present the tools included in the pre-defined set of plugins shipped with OSATE.

The following plug-ins are currently included with OSATE:

  • Weight (Mass) Analysis: analyzes the weight of components in an instance model.
  • Electrical Power Analysis: analyzes electrical power budgets against electrical power supply within the capacity of the electrical power transmission system.
  • Resource Budget Analysis for MIPS, RAM, ROM, and Network Bandwidth: compares resource budgets against capacities, in case of components bound to a resource, only for those budgets bound to a given resource.
  • Other SEI analysis plug-ins: analyzes models for given quality criteria (security, safety, performance, etc.).
