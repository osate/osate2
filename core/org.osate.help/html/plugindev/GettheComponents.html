2.1 Get the Components
If you do not already have Eclipse installed, it is easiest to download the single OSATE SDK distribution:
    • Download the OSATE SDK from the AADL homepage, http://www.aadl.info/. The current version bundles together Eclipse SDK 3.0.1, EMF SDK 2.0.1, and OSATE 1.0. The file name should be something like osate- 1.0-eclipse-SDK-3.0.1-emf-SDK-2.0.1- win32-05122005.zip.
If you are installing all the components piecemeal, you need to download the following distributions:
    • Download the Eclipse SDK—not the Runtime—from the Eclipse homepage, http://www.eclipse.org/. You need at least version 3.0. As of this writing, the current preferred version is release 3.0.1. The filename should be something like eclipse-SDK-3.0.1-win32.zip (for Windows) or eclipse-SDK-3.0.1-linux- gtk.zip (for Linux).
    • Download the EMF/SDO/XSD Software Development Kit (SDK) from the EMF homepage, http://www.eclipse.org/emf/. You need at least version 2.0. As of this writing, the current preferred version is release 2.0.1. The file name should be something like emf-sdo-xsd-SDK-2.0.1.zip.
    • Download the OSATE plug-in from the AADL homepage, http://www.aadl.info/. As of this writing, the current version is 1.0. The filename should be something like osate-1.0-05122005.zip.
