Get help and support

For users that have questions and want to get support for using AADL, you can get help on the official AADL wiki and also stay in touch with the community by posting on the community mailing lists.

Bug Report

If you want to report a bug, please go on our github account (on https://github.com/osate) and report a bug using our issue tracker. Please note that you will need to create a github account for using the issue tracker.
