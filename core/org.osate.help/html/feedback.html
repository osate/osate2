OSATE Help and Support

If you have questions and want to get support for using OSATE, or just want to stay in touch with the community you can join the OSATE Google group.

Bug Reporting

If you run into a problem with OSATE, please report it using our issue tracker. Please note that you will need to create a github account for using the issue tracker.

Please do not use the issue tracker to ask questions, that's what the Google group is for.
